# ALogs v1.0

Admin command logger for Gmod

This addon can log any command executed through **SAM** or **ULX**.

## Features

- Supports ULX and SAM admin mods
- SQLite and MySQL database support
- Multi-language support (English, Spanish)
  
## Preview

![ALogs Preview](https://raw.githubusercontent.com/zukkma/gmod-admin-logs/main/preview.png)

## Installation

### Using Git

Navigate to your server's addons directory and clone the repository:
```bash
cd garrysmod/addons
git clone https://github.com/zukkma/gmod-admin-logs.git
```

### Manual Installation

1. Download the latest release
2. Extract into `garrysmod/addons/`
3. Restart your server

## Config

Edit `alogs/lua/alogs/config.lua`

## Usage

Type `!alogs` in chat to open the logs menu

## Requirements

- **ULX** or **SAM** admin mod
- **[MySQLOO](https://github.com/FredyH/MySQLOO)** (optional, for MySQL support)
